# @libs/api-contract

Shared API contract using Zod schemas with autogenerated OpenAPI spec.

## Files

- `contract.ts` - Source of truth (Zod schemas + OpenAPI definitions)
- `openapi.json` - **AUTOGENERATED** - Do not edit manually

## Regenerate OpenAPI Spec

```bash
pnpm --filter api-contract build
```

Always regenerate after modifying `contract.ts`.
